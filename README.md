# Project 04 - Google Cloud Run Application

## 📖 Overview

This project demonstrates how to deploy a containerized Python web application using Google Cloud Run.

The application was packaged into a Docker container, built using Google Cloud Build, and deployed to Cloud Run as a fully managed serverless service.

---

## 🎯 Project Objectives

- Build a Flask web application
- Package the application using Docker
- Build a container image with Cloud Build
- Deploy the application to Cloud Run
- Access the application using its public URL

---

## ☁️ Google Cloud Services Used

- Cloud Run
- Cloud Build
- Artifact Registry (container image storage)
- Cloud Shell

5. Deployed the container image to Cloud Run.
